Disciplinas estudadas em front-end

HTML 5 é a linguagem padrão para criar páginas web. Ele estrutura todo o conteúdo e define elementos como títulos, parágrafos, links e imagens.

CSS estiliza páginas web, controlando layout, cores, fontes e espaçamento, tornando o conteúdo visualmente atraente e organizado.

JavaScript cria interatividade em páginas web, manipulando elementos HTML e CSS dinamicamente, tornando a experiência do usuário mais rica e responsiva.
